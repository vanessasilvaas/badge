## Coamndo Git
- git clone (clonar um repositorio)
- git branch (mostrar branch atual)
- git checkout -b (criar nova branch)
- git commit -m (salvar alterações)
- git status (verificar status da branch)
- git push origin  (enviar alterações ao repositório remoto)
- git add . (adiciona todos arquivos do path no stage changes)
- git add arquivo (adiciona um unico arquivo no stage changes)
